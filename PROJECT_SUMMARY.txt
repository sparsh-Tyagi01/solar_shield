                          🧠 AI MODELS

MODEL A: Storm Occurrence Predictor ⚡
├─ Algorithm: XGBoost Binary Classifier
├─ Purpose: Will storm occur in next 12-24 hours?
├─ Features: 6 (Bz, speed, density, gradients)
├─ Output: Yes/No + Probability + Risk Level
└─ Accuracy: ~87% expected

MODEL B: Storm Severity Predictor 📊
├─ Algorithm: LSTM Time-Series Neural Network
├─ Purpose: How severe will the storm be?
├─ Architecture: 64→32 LSTM units + Dense layers
├─ Output: Severity score (0-10) + Category
└─ Performance: MAE ~1.2, RMSE ~1.8

MODEL C: Impact Risk Classifier 🛰️
├─ Algorithm: Random Forest Multi-Label
├─ Purpose: Which systems will be affected?
├─ Categories: Satellites, GPS, Communication, Power Grid
├─ Output: Risk probability per system
└─ Accuracy: ~85% overall

                        🚀 QUICK START GUIDE
═══════════════════════════════════════════════════════════════════════

STEP 1: Setup Environment
    $ ./setup.sh

STEP 2: Activate Virtual Environment
    $ source venv/bin/activate

STEP 3: Install Dependencies (if not done by setup.sh)
    $ pip install -r requirements.txt

STEP 4: Test Components
    $ python test_components.py

STEP 5: Train Models (5-10 minutes)
    $ python backend/ml/train_pipeline.py

STEP 6: Start API Server
    $ uvicorn backend.main:app --reload

STEP 7: Access API
    🌐 API: http://localhost:8000
    📖 Docs: http://localhost:8000/docs
