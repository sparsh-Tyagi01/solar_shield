# SolarGuard 3D Frontend

Modern React-based frontend for SolarGuard 3D with real-time space weather monitoring and 3D visualizations.

## Features

- **Real-Time Dashboard**: Live solar wind metrics and geomagnetic activity
- **3D Earth Visualization**: Interactive magnetosphere with Three.js
- **Storm Prediction**: AI-powered forecasting display
- **Impact Analysis**: Infrastructure risk assessment
- **Historical Data**: Time-series charts and analysis
- **WebSocket Integration**: Real-time data streaming

## Installation

```bash
cd frontend
npm install
```

## Running

```bash
npm start
```

The app will run on [http://localhost:3000](http://localhost:3000)

## Tech Stack

- React 18
- TypeScript
- Three.js / React Three Fiber
- Recharts
- Tailwind CSS
- Framer Motion
- Socket.io Client
- Axios

## API Configuration

Backend API should be running on `http://localhost:8000`
